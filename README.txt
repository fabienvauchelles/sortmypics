The project has been developed in Java using Netbeans.

Features :
* find picture doubles
* sort by date and position
* retrieve Google Latitude history to match pictures which doesn't have a GPS tag
* use your Google address book to add a pretty title (eg: 2012-02-12 - Paris - Fabien home)
